
###Pole mass in terms of running masses and couplings

**M**      - is an **on-shell** mass

**m(mu)**  - runnig **MS** mass

**X_ij**   - functions of **MS** masses

**al(mu)** - EM running coupling

**as(mu)** - QCD running coupling

**`M = m (1 + as*X_01 + al*X_10 + al*as*X_11 + al^2*X_20 + ...)`**
